
1)    The module can be enabled and run on any standard Drupal 7 configuration.
2)    Have a normal menu item that triggers a page callback. The menu item can live in the standard navigation menu.
3)    The page callback should return a form implemented using the form API containing the following elements.
    A.    Text input field for a number.
    B.    A second text input field for a number.
    C.    A submit button
4)    When the form is submitted, it should return the sum of the 2 numbers in a standard Drupal message or elsewhere on the page.
5)    As a bonus, validate the form input to ensure the values are numeric.
